### 🎭 场景元素详解

#### 序章 - 星空梦幻
```
元素：40颗闪烁星星 + 2个旋转装饰圆形 + 律动装饰线
效果：twinkle 动画，星光闪烁
背景：深蓝梦幻渐变
```

#### 生平履历 - 书架意境
```
元素：3本浮动书籍 + 书架纹理 + 背景
效果：float-book 动画，书籍上下浮动
背景：棕色古朴渐变
```

#### 治学风骨 - 笔墨书房
```
元素：毛笔笔画装饰 + 水墨晕染 + 背景
效果：paint-flow/ink-spread 动画
背景：青色书房渐变
```

#### 科研丰碑 - 实验室
```
元素：3个电子轨道 + 原子核 + 2个试管 + 仪器
效果：orbit-rotate 轨道旋转，liquid-pulse 液体脉冲
背景：紫红科技渐变
```

#### 尾声 - 宇宙宏观
```
元素：60颗星星分布 + 2个星云 + 光束 + 宇宙背景
效果：nebula-breath 星云呼吸，beam-pulse 光束脉冲
背景：深蓝宇宙渐变
```

## 📝 内容管理

### 修改场景背景色
编辑 `pages/exhibit/exhibit.wxss`：
```css
.section-bg-2 { background: linear-gradient(...); }
.section-bg-3 { background: linear-gradient(...); }
```

### 修改云图节点
编辑 `pages/exhibit/exhibit.js`：
```javascript
eventClouds: [
  { id: 1, name: '名称', color: '#64c8ff', x: 0.5, y: 0.3, unlocked: false },
  ...
]
```

### 调整卷轴尺寸
编辑 `pages/exhibit/exhibit.wxss`：
```css
.scroll-item {
  width: 300rpx;  /* 调整宽度 */
  height: 420rpx; /* 调整高度 */
}
```
